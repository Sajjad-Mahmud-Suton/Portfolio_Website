# CV Download File

Please place your CV PDF file here with the name `cv.pdf`

This file will be downloaded when users click the "Download CV" button in the navigation bar.

## Instructions:

1. Convert your resume/CV to PDF format
2. Name the file `cv.pdf`
3. Place it in this directory (`assets/cv/`)

The CV download link is now configured to download `cv.pdf` from this location.
